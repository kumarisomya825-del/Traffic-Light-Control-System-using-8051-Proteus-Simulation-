# 🚦 Traffic Light Control System using 8051 (Proteus Simulation)

## 📌 Description
This project implements a traffic light control system using the AT89C51 microcontroller.  
The system is simulated in Proteus and demonstrates automatic switching of traffic signals.

---

## 🎯 Features
- Red, Yellow, Green LED control
- Two-side traffic signal system
- Automatic delay-based switching
- Implemented using 8051 Assembly/C

---

## ⚙️ Components Used
- AT89C51 Microcontroller
- LEDs (Red, Yellow, Green)
- Proteus ISIS software

---

## 🛠️ Tools & Technologies
- Keil uVision (for coding)
- Proteus ISIS (for simulation)
- coding in  8051 Assembly

---

## 🔄 Working Principle
The traffic lights change in the following sequence:

1. 🔴 Red ON → Others OFF  
2. 🟡 Yellow ON → Others OFF  
3. 🟢 Green ON → Others OFF
